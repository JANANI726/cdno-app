# CDNO - Context-Dependent Notification Optimizer


## Overview
CDNO is a context-aware notification ranking system designed to intelligently prioritize notifications based on user state and relevance.


The system filters and ranks alerts using contextual bias such as:
- Work Mode
- Sleep Mode
- Weekend Mode


It is designed as a high-fidelity frontend prototype with a local persistence layer using browser localStorage.


---


## Features
- Professional landing page
- Dashboard-based app interface
- 30+ Indian-based notification examples
- Priority ranking engine
- Filters by category and urgency
- Notification history tracking
- User profile/settings section
- Testimonials and contact CTA
- Fully responsive
- Built with HTML, CSS, JavaScript only
